## 一、市场背景

### 考试基本信息

| 项 | 内容 |
|----|------|
| 考试名称 | 中级会计专业技术资格考试 |
| 主管 | 财政部 + 人社部 |
| 科目 | 《中级会计实务》《财务管理》《经济法》 |
| 成绩管理 | 2 年内 3 科全部及格（60 分/100 分） |
| 考试时间 | 每年 9 月 |
| 报名条件 | 大专+5年/本科+4年/硕士+1年/博士 |
| 报名人数 | 约 200 万/年 |
| 通过率 | 整体约 10-15%，单科约 20-30% |

### 痛点分析

| # | 痛点 | 严重程度 | 说明 |
|---|------|----------|------|
| 1 | **时间不够** | ⭐⭐⭐⭐⭐ | 考生绝大多数在职，平均备考 3-6 个月，每天可用 1-2 小时 |
| 2 | **学完就忘** | ⭐⭐⭐⭐⭐ | 间隔复习做得差，学了后面忘前面，考前大脑空白 |
| 3 | **不知道重点** | ⭐⭐⭐⭐ | 教材动辄 300 页/科，分不清主次，平均用力导致核心考点没吃透 |
| 4 | **做题效率低** | ⭐⭐⭐⭐ | 已会的题反复做，不会的题只看答案不理解，无效刷题 |
| 5 | **2 年周期管理难** | ⭐⭐⭐⭐ | 第一年过了 2 科，第二年第 3 科没过，第一年全作废 |
| 6 | **没有学习节奏** | ⭐⭐⭐ | 缺少持续动力，容易半途而废（"有始无终"） |
| 7 | **在职没人教** | ⭐⭐⭐ | 网课时间不灵活，直播课跟不上，录播课没人答疑 |

### 竞品分析

| 竞品 | 模式 | 价格 | 优势 | 劣势 |
|------|------|------|------|------|
| **中华会计网校** | 录播课+题库 | ¥600-2000/科 | 名师多、内容全、品牌老 | 贵、课程长（100+小时）、互动差 |
| **东奥会计在线** | 录播课+题库 | ¥500-1800/科 | 轻松过关系列书强 | 同上，AI 能力弱 |
| **对啊网** | 直播+题库 | ¥300-1000/科 | 价格低、直播互动 | 师资一般、碎片化差 |
| **粉笔会计** | 题库+直播 | 免费-¥300 | 题库体验好、免费内容多 | 中级非主营、深度不够 |
| **233网校** | 录播+题库 | ¥300-800 | 价格适中 | 品牌弱、内容旧 |

**竞品共同缺陷**：
- ❌ 没有 AI 自适应学习路径（千人一面）
- ❌ 没有间隔重复记忆系统
- ❌ 没有薄弱点精准打击
- ❌ 没有学习节奏管理（无人督促）
- ❌ 答疑靠人工/论坛，等待时间长

## 三、产品定义

### 一句话定位

**"每天 1 小时，AI 帮你过中级"** — 基于间隔重复和自适应学习的会计中级备考助手。

### 核心差异化

| 传统网校 | 会过 |
|----------|------|
| 100 小时录播课，看完就忘 | AI 生成你的学习路径，只学该学的 |
| 千人一面 | 千人千面，根据你的薄弱点动态调整 |
| 做题看答案 | AI 讲题：为什么错、考什么点、怎么记住 |
| 学不学靠自己 | 学习节奏管理：每日推送、薄弱预警、2 年周期守护 |
| 课程 600-2000 元 | 免费+会员（¥99-199/季） |

### 不是什么

- ❌ 不是录播课平台（不做视频内容）
- ❌ 不是题库堆砌（不是把历年真题搬运过来）
- ❌ 不是 AI 聊天机器人（不是随意问问题的 ChatGPT）
- ✅ 是 **AI 学习教练** — 告诉你学什么、怎么学、记住了没

## 四、核心功能

### P0 — MVP（8 周交付）

#### 1. AI 学习路径 🎯

```
输入：目标科目 + 考试日期 + 每天可用时间
输出：每日学习计划（知识点 + 题目 + 复习）

规则：
- 优先高分考点（2/8 定律：20% 考点覆盖 80% 分值）
- 间隔重复：第 1/3/7/14/30 天自动安排复习
- 动态调整：做错了自动加练，做对了自动跳过
- 2 年周期守护：提醒"你第 1 年过了 2 科，第 2 年只剩 1 次机会"
```

#### 2. 智能刷题 📝

```
输入：选择科目/章节
输出：AI 筛选的题目（基于你的薄弱点）

规则：
- 先做 20 题诊断 → 找出薄弱章节
- 优先推薄弱章节的题
- 做错后：AI 讲解（不是只给答案）
  - 这道题考什么知识点
  - 你为什么选错（常见陷阱）
  - 类似题目怎么避坑
  - 相关知识点 3 句话回顾
- 做对 3 次同类题 → 标记掌握，不再推
```

#### 3. 间隔记忆卡 🧠

```
输入：知识点库（按科目/章节）
输出：每日记忆卡片推送

规则：
- 知识点拆成 1 张卡 = 1 个概念 + 1 道例题
- 间隔重复：1 天 → 3 天 → 7 天 → 14 天 → 30 天
- 记住了 → 拉长间隔
- 忘了 → 缩短间隔，重新来
- 每天推 15-20 张卡，5 分钟刷完
```

#### 4. 学习看板 📊

```
展示：
- 今日学习进度（做了多少题/记了多少卡/学了多久）
- 薄弱章节排行（红色=未掌握，黄色=部分掌握，绿色=掌握）
- 距考试倒计时 + 剩余知识点估算
- 2 年成绩守护（第一年过了哪些科，第二年还有几次机会）

目的：让用户"知道自己在哪"，消除盲目焦虑
```

### P1 — 第二版（上线后迭代）

| 功能 | 说明 |
|------|------|
| AI 答疑 | 不会的知识点直接问 AI，结合教材原文回答 |
| 错题本 | 自动归集错题，按知识点分类，到期自动推送复习 |
| 模拟考试 | 仿真模考 + AI 评分 + 薄弱点分析 |
| 学习小组 | 3-5 人打卡群，互相督促 |
| 碎片音频 | 知识点 3 分钟音频，通勤路上听 |

### P2 — 远期

| 功能 | 说明 |
|------|------|
| 初级会计 | 扩展到初级会计职称 |
| CPA | 扩展到注册会计师 |
| 企业版 | 培训机构/企业内训白标 |
| AI 出题 | 根据最新考纲自动生成仿真题 |

- ❌ 模拟考试
- ❌ 社交功能

### 技术选型方向

| 层 | 选择 | 理由 |
|----|------|------|
| 前端 | 微信小程序 + H5 | 目标用户在微信生态，小程序获客成本最低 |
| 后端 | Supabase | 托管优先，月成本 < ¥200 |
| AI | Claude API | 生成学习路径 + 讲题 + 记忆卡 |
| 内容存储 | Supabase Postgres | 题库 + 知识点 + 用户进度 |

## 七、验证计划

### 2 周内验证（投入最小）

1. **痛点验证** — 在小红书/知乎发 3 篇帖子，看阅读量和评论
   - 帖子主题："中级备考每天 1 小时够吗？""AI 能帮你过中级吗？"
   - 指标：阅读 > 1000，评论 > 20

2. **Demo 验证** — 做一个可运行的 Demo（微信小程序 1 科 50 题）
   - 让 10 个目标用户试用
   - 收集 NPS 和付费意愿

3. **定价验证** — 在帖子/Demo 中测试价格敏感度
   - ¥49/季 vs ¥99/季 vs ¥199/年
   - 问："如果这个产品卖 X 元，你会买吗？"

### 验证判断

| 信号 | 通过标准 | 不通过 |
|------|----------|--------|
| 痛点真实 | 10 人中 7 人认同核心痛点 | < 5 人认同 |
| Demo 价值 | 10 人中 6 人愿意试用 | < 4 人愿意 |
| 付费意愿 | 10 人中 3 人愿付费 | 0 人愿付费 |
